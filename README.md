# 妈妈的专属批发记账簿 (Mama's Ledger)

这是一个为猪肉批发行业量身定制的轻量级记账 Web 应用。旨在解决传统手写账本难翻阅、难对账、难汇总的痛点。

### ✨ 核心功能
* **多品类动态录入**：支持前腿肉、五花肉、白条等 10+ 种货品，独立计算重量、单价、减称。
* **老板思维盈亏核算**：基于“应收 - 本金”逻辑，实时计算生意利润。
* **全账单档案检索**：输入客户名，一键调取历史所有往来账单与累计欠款。
* **专业财务导出**：
    * **日表**：详尽的货品明细，用于日常对账。
    * **月表**：汇总日期、本金、盈亏、欠款，用于经营分析。
* **容错保护**：支持二次编辑、删除确认，内置“后悔药”机制。

### 🛠️ 技术栈
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
* **Storage**: Browser LocalStorage (数据存储在本地，不上传服务器，保护隐私)
* **Plugins**: [SheetJS](https://sheetjs.com/) (支持 Excel 导出)

### 📱 使用说明
1.  **添加桌面**：在手机浏览器打开网址，点击“添加到主屏幕”，即可像 App 一样使用。
2.  **每日备份**：建议每天收摊后点击“导出 Excel”发送至微信，实现双重数据保障。
